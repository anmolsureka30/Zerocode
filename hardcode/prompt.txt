Build a modern, responsive e-commerce web application using React, TypeScript, and Tailwind CSS. Include a home page with a hero section, featured products, category navigation, and newsletter signup. Enable product browsing with category filtering, sorting options, and a responsive grid layout. Provide detailed product pages with quantity selection and related products. Implement a shopping cart allowing users to add, update, or remove items, clear the cart, and view a summary with subtotal, shipping, tax, and total. Use React Router for navigation, Context API for cart state management, and Shadcn UI for accessible components.
